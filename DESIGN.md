# higeball-quest 設計書 v0.2

## ゲーム概要

- **タイトル**: HIGEBALL QUEST
- **ジャンル**: 2D見下ろし型RPG
- **プラットフォーム**: スマホブラウザ（縦持ち専用）
- **プレイ時間**: 第1章プロトタイプは15〜30分
- **トーン**: ほのぼの系、第二の人生、旧友探し

## 技術スタック

- 言語: TypeScript
- フレームワーク: Phaser 3
- ビルド: Vite
- 公開: GitHub Pages

## 画面・操作

- 画面サイズ: 360×640（縦持ち）
- 操作: 仮想十字キー＋A/Bボタン／キーボード／スワイプ併用
- ドット絵風の簡易グラフィックは Phaser Graphics で描画する。

## ストーリー 第1章

60歳でシステムエンジニアをリタイヤした `yos` は、第二の人生をどうするか迷っていた。まずは10年会っていない旧友を探すところから旅が始まる。

最初に探すのは、一番頼りになる `もじさん`。チバの村の住人に話しかけると、むかしヨツビシという会社の同僚だったこと、退職後はヒョーゴの村で農家をやっていることを聞く。yos は電車でヒョーゴの村へ向かう。

ヒョーゴの村では、もじさんが畑を充実させる新しい野菜を探しに山へ行ったまま戻っていないと聞く。装備を整え、山頂へ向かうとボス `オニオンJK` が現れる。yos ともじさんは2人で協力して倒す。

撃破後、もじさんは正式に仲間になり、次の仲間探しの旅に出る。続きは後続アップデートで追加する。

## パーティ構成

| 名前 | ジョブ | 役割 |
|---|---|---|
| yos | ゲーマス | `じゅもん` を使う。初期技はHP小回復の `ホイミ` |
| もじさん | 百姓貴族 | `ベジタブル` を使う。初期技は1.5倍ダメージの `オニオンソード` |

## 戦闘

- SFC FF6風の左にモンスター、右に味方のスタイル。
- ATBではなくコマンド選択式。
- コマンドはキャラクターごとに異なる。
  - yos: `たたかう` / `じゅもん` / `ぼうぎょ` / `もちもの`
  - もじさん: `たたかう` / `ベジタブル` / `ぼうぎょ` / `もちもの`
- ダメージ、回復は該当キャラ上に数字で表示する。

## マップ構成

- チバの村: 旧友の手がかりを聞く拠点。
- ヒョーゴの村: もじさんの現在地を探る拠点。
- 山道ふもと、山道中腹、山頂: ランダムエンカウントあり。山頂にオニオンJK。
