# HIGEBALL QUEST 実装仕様

このファイルは、AIエージェントや開発者が現在のゲーム仕様を読み取り、同等のプロトタイプを再実装・改修できる粒度で整理する。実装変更時はこのファイルも必ず更新する。

## 目的

- `DESIGN.md` の構想をもとにした、スマホ縦持ちブラウザ向け2D見下ろし型RPG。
- 現在は第1章プロトタイプ段階。チバの村、ヒョーゴの村、山道3層、もじさん加入、ランダム戦闘、ボス戦、第1章完了まで通しで遊べる。
- 続きの仲間探しを後から追加できるよう、進行フラグと保存形式は拡張前提にしている。

## 技術構成

- 言語: TypeScript
- ゲームエンジン: Phaser 3
- ビルド: Vite
- エントリ: `index.html` から `/src/main.ts` を読み込む
- スタイル: `styles.css`
- 保存: `localStorage`
- 公開: GitHub Pages。`main` への push で GitHub Actions が `dist/` を公開する。

## 画面構成

- 全体表示はスマホ縦持ち想定の `360x640`。
- 上部 `360x472` が Phaser のゲーム画面。
- 下部 `168px` がHTMLのバーチャルパッド領域。
- ゲーム画面と操作領域は重ねない。
- 誤操作を避けるため、Bボタンは右方向ボタンから十分に離し、Aボタンの左上側へ独立配置する。
- 起動時はタイトル画面を表示する。`はじめから` / `つづきから` を上下で選び、Aで決定する。Bでもロードスロット選択を開く。
- フィールドはFF6風のワールドマップとして扱い、村・街・ダンジョンの入口アイコンを配置する。アイコンの上に乗ると中へ入れる。
- フィールド上部HUDには、現在地、所持金、消耗品数、各メンバーの名前、Lv、HPを表示する。

## 操作

- 仮想十字キー: 移動、メニュー選択、戦闘コマンド選択。
- Aボタン: 決定、会話送り、調べる、戦闘コマンド実行。
- Bボタン: キャンセル、メニュー表示、戦闘コマンド送り。
- キーボード:
  - 矢印キー: 移動/選択
  - `Z` / `Enter` / `Space`: A
  - `X` / `Escape`: B
- スワイプ: フィールド移動補助。
- ランダムエンカウントまたはボス戦突入の短い演出中は入力を受け付けない。

## ストーリー

- 60歳でシステムエンジニアをリタイヤした `yos` が、第二の人生をどうするか迷い、10年会っていない旧友を探し始める。
- 最初の目的は、一番頼りになる旧友 `もじさん` を探すこと。
- チバの村の元同僚から、もじさんはヨツビシ時代の同僚で、退職後はヒョーゴの村で農家をやっていると聞く。
- 電車でヒョーゴの村へ向かう。
- ヒョーゴの村では、もじさんが畑を充実させる新しい野菜を探して山へ行ったまま帰らないと聞く。
- 山頂で `オニオンJK` と遭遇し、yos ともじさんが協力して倒す。
- 撃破後、もじさんは正式に仲間になり、次の仲間探しへ出る。現在は「第1章 もじさん加入 完」まで。
- ボス戦で全滅した場合、もじさんは一時参加から離脱し、再挑戦時に山頂で再合流する。
- 会話文が `話者: 本文` の形式の場合、メッセージウィンドウに話者名と簡易ポートレートを表示する。
- 会話中はフィールドを少し暗くし、メッセージ送りが必要な場合はウィンドウ右下に小さな続きマーカーを表示する。
- NPC会話は、加入状況や進行フラグに応じて分岐する。元同僚、農家、宿屋、案内役などは状況で台詞が変わる。

## マップ

- `フィールド` はオーバーワールド。村・街・ダンジョンの入口アイコンがあり、そこへ入ると各内部マップへ遷移する。
- マップは共通タイルに加えて、場所ごとの探索演出を重ねて描画する。
- チバの村は南側に駅/線路の視覚ヒントを表示し、ヒョーゴの村は畑の畝と野菜の点描で農村感を出す。
- 山道は階層ごとに暗さ、岩、ひび割れを重ね、山頂はボス周辺に淡い光と風の線を描く。
- 案内NPC、出口、未開封宝箱は、それぞれマーカーや発光で自然に目立たせる。
- 村・森・ダンジョンごとに薄い色味のティントを重ね、同じタイルでも見え方を変える。
- フィールド上では、チバの村、ヒョーゴの村、山道ふもとの入口をアイコンとして見せる。

### チバの村

- エンカウントなし。
- 宿屋、武器屋、防具屋、道具屋、教会、元同僚、駅の案内がある。
- 元同僚に話すと `heardMojisanLead` フラグが立ち、ヒョーゴの村へ進める。
- 南へ進むとフィールドへ戻る。

### ヒョーゴの村

- エンカウントなし。
- 農家、村人、山の案内がある。
- 農家に話すと `heardMojisanMountain` フラグが立ち、山道へ進める。
- 北へ戻るとフィールドへ戻る。

### 山道

- 3層構成。
- ランダムエンカウントあり。
  - 山道ふもと: `0.16`
  - 山道中腹: `0.18`
  - 山頂: `0.20`
- 山頂の奥にボス「オニオンJK」。
- ボス接触時、フィールド上でもじさんがオニオンJKの横に立ち、一時的に戦闘参加する。会話イベント後にボス戦へ入る。
- ボス撃破後はヒョーゴの村へ戻り、もじさんが正式加入する。
- 各層に `まほうの水` の宝箱がある。

## パーティ

初期メンバーは `yos` のみ。第1章ボス戦から `もじさん` が一時参加し、ボス撃破後に正式加入する。

| 名前 | 初期Lv | ジョブ | 役割 |
|---|---:|---|---|
| yos | 1 | ゲーマス | 回復じゅもんを使えるバランス型 |
| もじさん | 5 | 百姓貴族 | 高HP・高攻撃。ベジタブル技を使う |

今後のアップデートで追加メンバーを増やす前提のため、保存形式は複数加入フラグに対応している。

## キャラクター表現

- フィールド上では、加入済みの仲間が主人公の後ろに追従表示される。
- 移動中は主人公と追従メンバーの足・腕に短い歩行アニメーションを入れる。
- NPCは控えめな待機揺れを持ち、完全な静止表示にならないようにする。
- 戦闘中の味方は、アクティブ時に軽い待機揺れを入れる。
- キャラクターごとの差を出すため、yos は眼鏡/小物、もじさんは農家風の帽子と野菜モチーフを持つ。

## 成長

- 経験値とレベルはキャラクターごとに保持する。
- 戦闘勝利時、参加中メンバーそれぞれに同じEXPを加算する。
- レベルアップ時は対象キャラクターの最大HP/MP、力、体力、魔力、素早さ、回避などを上げる。
- レベルアップしたキャラクターのみ、戦闘終了時にHP/MPを全回復する。
- 既存セーブデータに個別EXPや新ステータスがない場合、ロード時に現在Lvや既存の攻撃/防御値から補完する。
- 現在の累計EXPしきい値:
  - Lv1: 0
  - Lv2: 20
  - Lv3: 48
  - Lv4: 88
  - Lv5: 140
  - Lv6: 210
  - Lv7: 300

## ステータス

- キャラクターはHP/MPに加え、FF6風の基本ステータスを持つ。
- `力`: 通常攻撃や物理系スキルの基礎値。装備の攻撃補正を加えてダメージ計算に使う。
- `体`: 防御の基礎値。防具の防御補正を加えて被ダメージ計算に使う。
- `速`: 行動順や将来の速度差演出に使う予定の値。
- `魔`: 回復/魔法系スキルの成長に使う予定の値。
- `回`: 回避率や回避演出に使う予定の値。

## スキル

- yos のスキル系統は `じゅもん`。テレビゲーム由来の技として拡張する。
  - 初期: `ホイミ`。MP4消費。HPが最も減っている味方を小回復する。
  - 成長: `ベホイミ`、`メガホイミ` をレベルで習得し、稀にイベントで追加の回復技を覚える。
- もじさんのスキル系統は `ベジタブル`。野菜にちなんだ技として拡張する。
  - 初期: `オニオンソード`。MP3消費。通常攻撃相当の1.5倍ダメージ。
  - 成長: `オニオンラッシュ`、`オニオンブラスト` をレベルで習得し、稀にイベントで追加の野菜技を覚える。
- 技はレベルアップとイベント習得で増え、戦闘中のサブウィンドウに習得済みのものだけを表示する。

## アイテム

- `やくそう`: HPを28回復。戦闘中、使用者のHPが減っている場合に使う。
- `まほうの水`: MPを12回復。戦闘中、HP回復が不要かつ使用者のMPが減っている場合に使う。
- `もちもの` コマンドは現状サブメニューなし。条件に応じて自動で使う。

## ショップ/装備

- 村のNPCからショップを開く。
- 購入時は所持金を消費し、道具は `items`、武器/防具は `inventory` に追加する。
- ショップでは選択中商品の所持数と装備可能者を表示する。
- ショップ購入後は店メニューを閉じず、続けて購入できる。
- Bメニューに `そうび` と `もちもの` がある。
- `そうび` は `全員 最強装備` と個別キャラクター選択を持つ。
- 個別装備では、対象キャラクターが装備可能な所持中の武器/防具から選んで変更できる。
- 攻撃/防御計算には装備補正を含める。
- もじさんは加入時に `オニオンソード` を所持する。

## 戦闘

- コマンド選択式ターン制。
- ATBではない。
- 全滅時はチバの村で全回復して再開する。所持金、経験値、アイテムは失わない。
- ボス戦で全滅した場合、一時参加中のもじさんはパーティから離脱し、加入フラグも未加入へ戻る。再挑戦時は山頂イベントで再び一時参加する。
- 敵は左、味方は右に配置。
- 戦闘コマンドはキャラクターごとに変わる。
  - yos: `たたかう` / `じゅもん` / `ぼうぎょ` / `もちもの`
  - もじさん: `たたかう` / `ベジタブル` / `ぼうぎょ` / `もちもの`
- `じゅもん` と `ベジタブル` は、選択後にサブウィンドウを開き、習得済みの技名を選んで実行する。
  - 現在の `じゅもん`: `ホイミ`
  - 現在の `ベジタブル`: `オニオンソード`
- スキルサブウィンドウは、技名、必要MP、効果の短い説明を表示する。Bでコマンド選択へ戻る。
- MPが不足しているスキルは実行できない。ターンは消費せず、サブウィンドウを開いたまま不足メッセージを表示する。
- `ぼうぎょ` は次に対象になった敵攻撃のダメージを半減する。
- 味方行動時は対象へ一歩踏み込む短い演出を入れる。テンポを落とさないため、移動は約0.26秒以内に収める。
- 味方行動後と敵行動後に短い待ちを入れ、メッセージと演出が読めるテンポにする。
- 戦闘開始、味方の手番、敵の手番は短い cue 表示で区切る。
- 行動処理中はコマンドカーソル入力を受け付けず、戦闘メニューに `行動中` を表示する。
- 敵HPバーと味方HPバーを表示。
- コマンド選択中はハイライトを表示。
- ダメージ/回復は対象付近にフロートテキストで表示。
- 敵は種類ごとの行動パターンを持つ。通常攻撃だけでなく、強攻撃、弱攻撃、回復などを使う。
- スライム系は軽攻撃や自己回復、こうもり系はMP吸収、ナイト系は連撃、メイジ系はMPを削る呪文を使う。
- ボスは専用行動として `玉ねぎ審査`、`涙目スライス`、低HP時の自己回復を使う。
- 戦闘下部UIはSFC FF6風の青いウィンドウを基調にする。
- 戦闘メッセージは直近1件を表示し、枠内で3行まで折り返す。長めの敵行動メッセージでも文字が枠外へ切れにくいよう、文字サイズと行間を調整している。
- `じゅもん` / `ベジタブル` のサブウィンドウ表示中は、背後のステータス文字を非表示にし、技名と説明が重ならず読めるようにする。
- ランダムエンカウントとボス戦突入時は、約0.22秒の短い画面フラッシュ/斜線演出を挟んでから戦闘画面へ切り替える。
- 戦闘中はフィールド用HUDを表示しない。

## 戦闘演出

- 通常攻撃: 斬撃エフェクト、敵フラッシュ、軽い画面揺れ。
- じゅもん/回復/道具: 緑系の粒子エフェクト。
- ベジタブル: 斬撃エフェクトと専用メッセージ。
- 被弾: 味方フラッシュ、軽い画面揺れ。
- 味方キャラと通常敵は Phaser Graphics で描き分ける。

## 敵

### 山道

- まゆげスライム: HP26、攻8、守2、12EXP、8G。弱めの体当たりを使う。
- ヒゲこうもり: HP32、攻10、守3、14EXP、10G。急降下攻撃を使う。
- ねぐせナイト: HP48、攻14、守5、24EXP、18G。力を込めた斬撃を使う。
- うすげメイジ: HP40、攻16、守4、28EXP、20G。呪文攻撃を使う。

### ボス

- オニオンJK: HP480、攻44、守15、240EXP、260G
- Lv7程度まで上げ、装備と道具を整えて倒す想定。

## セーブ/ロード

- Bメニューから `記録する` / `つづきから` / `そうび` / `もちもの` / `ステータス` を選択。
- 3スロット制。
- セーブ/ロードスロットには、空き状態または保存済みの場所、先頭メンバーLv、進行状況を表示する。
- 保存対象:
  - 現在マップ
  - 主人公位置
  - パーティ状態
  - 加入済みフラグ
  - 進行フラグ
  - アイテム
  - 武器/防具インベントリ
  - 所持金
  - キャラクターごとの経験値/レベル/ステータス
  - 第1章完了フラグ
  - 宝箱開封状態
  - 保存日時

## 音声

- 外部音源ファイルは使わず、Web Audio APIで簡易BGMとSEを生成する。
- スマホブラウザの自動再生制限に対応するため、最初のキー入力またはタップでAudioContextを初期化/再開する。
- BGMは場面ごとに自動切替する。
- SEは決定、キャンセル、移動/カーソル、攻撃、魔法、回復、宝箱、階段/出口で鳴る。

## 現在の制約

- スプライト画像はまだ使わず、Phaser Graphics で疑似ドット絵を描いている。
- 音声は簡易実装段階。音色は生成音で、音源ファイルやミキサーUIはまだない。
- ショップ購入UIと装備UIは簡易実装段階。売却、装備品の受け渡し、購入前の確認ダイアログはまだない。
- `もちもの` コマンドはサブメニューなし。
- 戦闘対象選択は未実装。敵は1体のみ。
- マップは固定画面でスクロールしない。
- スキルのレベルアップ習得/イベント習得は拡張予定。
- 第2章以降の仲間探しは未実装。
