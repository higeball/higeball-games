# HIGEBALL QUEST 実装仕様

このファイルは、AIエージェントや開発者が現在のゲーム仕様を読み取り、同等のプロトタイプを再実装・改修できる粒度で整理する。実装変更時はこのファイルも必ず更新する。

## 目的

- `DESIGN.md` の構想をもとにした、スマホ縦持ちブラウザ向け2D見下ろし型RPG。
- 現在は第1章プロトタイプ段階。チバの村、ヒョーゴの村、山道3層、もじさん加入、ランダム戦闘、ボス戦、第1章完了まで通しで遊べる。
- 続きの仲間探しを後から追加できるよう、進行フラグと保存形式は拡張前提にしている。

## 技術構成

- 言語: TypeScript
- ゲームエンジン: Phaser 3
- ビルド: Vite
- エントリ: `index.html` から `/src/main.ts` を読み込む
- スタイル: `styles.css`
- 保存: `localStorage`
- 公開: GitHub Pages。`main` への push で GitHub Actions が `dist/` を公開する。

## 画面構成

- 全体表示はスマホ縦持ち想定の `360x640`。
- 上部 `360x472` が Phaser のゲーム画面。
- 下部 `168px` がHTMLのバーチャルパッド領域。
- ゲーム画面と操作領域は重ねない。
- 誤操作を避けるため、Bボタンは右方向ボタンから離してやや上寄せに配置する。
- 起動時はタイトル画面を表示する。`はじめから` / `つづきから` を上下で選び、Aで決定する。Bでもロードスロット選択を開く。

## 操作

- 仮想十字キー: 移動、メニュー選択、戦闘コマンド選択。
- Aボタン: 決定、会話送り、調べる、戦闘コマンド実行。
- Bボタン: キャンセル、メニュー表示、戦闘コマンド送り。
- キーボード:
  - 矢印キー: 移動/選択
  - `Z` / `Enter` / `Space`: A
  - `X` / `Escape`: B
- スワイプ: フィールド移動補助。

## ストーリー

- 60歳でシステムエンジニアをリタイヤした `yos` が、第二の人生をどうするか迷い、10年会っていない旧友を探し始める。
- 最初の目的は、一番頼りになる旧友 `もじさん` を探すこと。
- チバの村の元同僚から、もじさんはヨツビシ時代の同僚で、退職後はヒョーゴの村で農家をやっていると聞く。
- 電車でヒョーゴの村へ向かう。
- ヒョーゴの村では、もじさんが畑を充実させる新しい野菜を探して山へ行ったまま帰らないと聞く。
- 山頂で `オニオンJK` と遭遇し、yos ともじさんが協力して倒す。
- 撃破後、もじさんは正式に仲間になり、次の仲間探しへ出る。現在は「第1章 もじさん加入 完」まで。

## マップ

### チバの村

- エンカウントなし。
- 宿屋、武器屋、防具屋、道具屋、教会、元同僚、駅の案内がある。
- 元同僚に話すと `heardMojisanLead` フラグが立ち、ヒョーゴの村へ進める。
- 南へ進むと電車でヒョーゴの村へ移動する。

### ヒョーゴの村

- エンカウントなし。
- 農家、村人、山の案内がある。
- 農家に話すと `heardMojisanMountain` フラグが立ち、山道へ進める。
- 北へ戻るとチバの村、南へ進むと山道ふもと。

### 山道

- 3層構成。
- ランダムエンカウントあり。
  - 山道ふもと: `0.16`
  - 山道中腹: `0.18`
  - 山頂: `0.20`
- 山頂の奥にボス「オニオンJK」。
- ボス接触時、もじさんが一時的に戦闘参加し、会話イベント後にボス戦へ入る。
- ボス撃破後はヒョーゴの村へ戻り、もじさんが正式加入する。
- 各層に `まほうの水` の宝箱がある。

## パーティ

初期メンバーは `yos` のみ。第1章ボス戦から `もじさん` が加入する。

| 名前 | ジョブ | 役割 |
|---|---|---|
| yos | ゲーマス | 回復じゅもんを使えるバランス型 |
| もじさん | 百姓貴族 | 高HP・高攻撃。ベジタブル技を使う |

今後のアップデートで追加メンバーを増やす前提のため、保存形式は複数加入フラグに対応している。

## 成長

- 経験値はパーティ共通。
- 全員同じレベルで成長する。
- レベルアップ時は全員の最大HP/MP、攻撃、防御を上げる。
- レベルアップが発生した戦闘終了時、HP/MPを全回復する。
- 現在の累計EXPしきい値:
  - Lv1: 0
  - Lv2: 20
  - Lv3: 48
  - Lv4: 88
  - Lv5: 140
  - Lv6: 210
  - Lv7: 300

## スキル

- yos のスキル系統は `じゅもん`。テレビゲーム由来の技として拡張する。
  - 初期: `ホイミ`。MP4消費。HPが最も減っている味方を小回復する。
- もじさんのスキル系統は `ベジタブル`。野菜にちなんだ技として拡張する。
  - 初期: `オニオンソード`。MP3消費。通常攻撃相当の1.5倍ダメージ。
- 将来、スキルはレベルアップとイベント習得で増やせるようにする。

## アイテム

- `やくそう`: HPを28回復。戦闘中、使用者のHPが減っている場合に使う。
- `まほうの水`: MPを12回復。戦闘中、HP回復が不要かつ使用者のMPが減っている場合に使う。
- `もちもの` コマンドは現状サブメニューなし。条件に応じて自動で使う。

## ショップ/装備

- 村のNPCからショップを開く。
- 購入時は所持金を消費し、道具は `items`、武器/防具は `inventory` に追加する。
- ショップでは選択中商品の所持数と装備可能者を表示する。
- ショップ購入後は店メニューを閉じず、続けて購入できる。
- Bメニューに `そうび` と `もちもの` がある。
- `そうび` は `全員 最強装備` と個別キャラクター選択を持つ。
- 個別装備では、対象キャラクターが装備可能な所持中の武器/防具から選んで変更できる。
- 攻撃/防御計算には装備補正を含める。
- もじさんは加入時に `オニオンソード` を所持する。

## 戦闘

- コマンド選択式ターン制。
- ATBではない。
- 全滅時はチバの村で全回復して再開する。所持金、経験値、アイテムは失わない。
- 敵は左、味方は右に配置。
- 戦闘コマンドはキャラクターごとに変わる。
  - yos: `たたかう` / `じゅもん` / `ぼうぎょ` / `もちもの`
  - もじさん: `たたかう` / `ベジタブル` / `ぼうぎょ` / `もちもの`
- `ぼうぎょ` は次に対象になった敵攻撃のダメージを半減する。
- 味方行動後と敵行動後に短い待ちを入れ、メッセージと演出が読めるテンポにする。
- 行動処理中はコマンドカーソル入力を受け付けず、戦闘メニューに `行動中` を表示する。
- 敵HPバーと味方HPバーを表示。
- コマンド選択中はハイライトを表示。
- ダメージ/回復は対象付近にフロートテキストで表示。
- 戦闘下部UIはSFC FF6風の青いウィンドウを基調にする。
- 戦闘メッセージは直近1件を表示し、枠内で2行まで折り返す。
- 戦闘中はフィールド用HUDを表示しない。

## 戦闘演出

- 通常攻撃: 斬撃エフェクト、敵フラッシュ、軽い画面揺れ。
- じゅもん/回復/道具: 緑系の粒子エフェクト。
- ベジタブル: 斬撃エフェクトと専用メッセージ。
- 被弾: 味方フラッシュ、軽い画面揺れ。
- 味方キャラと通常敵は Phaser Graphics で描き分ける。

- ヒゲこうもり: HP32、攻10、守3、14EXP、10G

## セーブ/ロード

- Bメニューから `記録する` / `つづきから` / `そうび` / `もちもの` / `ステータス` を選択。
- 3スロット制。
- セーブ/ロードスロットには、空き状態または保存済みの場所、先頭メンバーLv、進行状況を表示する。
- 保存対象:
  - 現在マップ
  - 主人公位置
  - パーティ状態
  - 加入済みフラグ
  - 進行フラグ
  - アイテム
  - 武器/防具インベントリ
  - 所持金
  - 経験値
  - 第1章完了フラグ
  - 宝箱開封状態
  - 保存日時

## 音声

- 外部音源ファイルは使わず、Web Audio APIで簡易BGMとSEを生成する。
- スマホブラウザの自動再生制限に対応するため、最初のキー入力またはタップでAudioContextを初期化/再開する。
- BGMは場面ごとに自動切替する。
- SEは決定、キャンセル、移動/カーソル、攻撃、魔法、回復、宝箱、階段/出口で鳴る。

## 現在の制約

- スプライト画像はまだ使わず、Phaser Graphics で疑似ドット絵を描いている。
- 音声は簡易実装段階。音色は生成音で、音源ファイルやミキサーUIはまだない。
- ショップ購入UIと装備UIは簡易実装段階。売却、装備品の受け渡し、購入前の確認ダイアログはまだない。
- `もちもの` コマンドはサブメニューなし。
- 戦闘対象選択は未実装。敵は1体のみ。
- マップは固定画面でスクロールしない。
- スキルのレベルアップ習得/イベント習得は拡張予定。
- 第2章以降の仲間探しは未実装。
